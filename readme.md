# 死因別死亡率の推移（1950–2024）

## 1. アプリ概要

本アプリは、e-Stat（政府統計ポータルサイト）で公開されている統計データを用いて、日本における死因別死亡率の推移を可視化するWebアプリです。

1950年から2024年までの各年について、死因上位10位の死亡率（人口10万人対）を対象とし、年ごとの比較や、特定の死因に着目した長期的な変化を確認できます。

## 2. アプリの目的

* 日本の死因構造が時代とともにどのように変化してきたかを把握する
* 年別・死因別の死亡率を比較し、傾向を視覚的に理解する

## 3. 使用データについて

* **統計表名**：死因順位別にみた年次別死亡率（人口10万対）
* **データ出典**：e-Stat（政府統計ポータルサイト）
* **対象期間**：1950年〜2024年
* **内容**：各年の死因上位10位とその死亡率
* **単位**：人口10万人対
* **データ形式**：CSV
* **データ取得日(更新日)**：2025年09月16日

### 各列の説明

* 年次：対象となる年
* 第1位〜第10位：その年の死因順位
* 第1位死亡率〜第10位死亡率：対応する死因の死亡率（人口10万人対）

## 4. アプリの使い方

画面左側のサイドバーから表示方法を選択します。

### 年別にみる

* 年を1つ選択すると、その年における死因上位10位と死亡率を表示します
* 棒グラフにより、死因ごとの死亡率を比較できます
* 表示切替により、数値データを表として確認することも可能です
* 最大・最小の死亡率と対応する死因が表示されます

### 死因別にみる

* 死因を1つ選択すると、1950年から2024年までの死亡率の推移を確認できます
* 折れ線グラフにより、長期的な変化や傾向を把握できます
* 最大値・最小値・平均値といった基本統計量が表示されます

## 5. 未使用UI部品

* st.metric：死亡率の最大値・最小値・平均値を強調表示するために使用

## 6. 可視化の工夫・ポイント

* 年別表示では棒グラフを用い、同一年内での死因比較をしやすくした
* 死因別表示では折れ線グラフを用い、時系列変化を直感的に理解できるようにした
* 数値情報とグラフをタブで分離し、目的に応じて閲覧できる構成にした
